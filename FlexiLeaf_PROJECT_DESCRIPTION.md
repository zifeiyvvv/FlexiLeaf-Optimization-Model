# FlexiLeaf Project Description

## One-sentence value proposition

FlexiLeaf turns Hong Kong open data and a flexible photovoltaic deployment blueprint into a traceable urban solar-planning, digital-twin and community-microgrid optimisation platform.

## English description

FlexiLeaf is an open-data-driven urban solar planning and community microgrid optimisation platform designed for Hong Kong’s dense built environment. It connects Hong Kong Observatory weather and solar data, Lands Department location services and CSDI building geometry to identify underused roofs, facades and public-infrastructure surfaces. The platform estimates photovoltaic potential, generates transparent building-load and solar scenarios, compares a grid-only baseline with roof PV, FlexiLeaf facade deployment, battery storage and aggregated vehicle-to-grid resources, and exports reproducible CSV, GeoJSON and JSON evidence. A fixed 2025 design-basis digital-twin case contains 8,760 hourly records and 15 scenario/configuration combinations. Under the recorded design assumptions, the full microgrid scenario produces approximately 2,505 MWh of simulated annual photovoltaic electricity and reduces simulated peak grid demand by 29.4% relative to the grid-only baseline. These results are prospective model outputs, not measured field performance.

## 中文描述

FlexiLeaf是一個面向香港高密度城市環境的開放數據驅動光伏規劃與社區微電網優化平台。系統整合天文台太陽輻射與天氣資料、地政總署地點搜尋及CSDI建築幾何數據，識別屋頂、建築立面、天橋及公共設施的潛在光伏表面，並建立可追溯的負荷、發電、儲能及車網互動情景。固定的2025設計基準數字孿生案例包含8,760小時數據與15組情景比較；在公開列明的設計假設下，完整微電網方案的模擬年光伏發電量約為2,505 MWh，模擬峰值電網負荷較僅電網基準降低29.4%。所有數字均為前瞻性模擬結果，並非實地運行表現。

## Repository description (short)

Open-data urban solar planning, annual digital-twin simulation and battery/V2G microgrid optimisation for Hong Kong.
